# pydevice2mqtt

Python class library for generating and sending data to the Home Assistant via
MQTT (AutoDiscovery) of data from devices

## Build and Install
### build wheel: 

    # from top folder: 
    pip install -e .
    flit build

### install wheel:
       
    # inside dist folder:
    pip install hass_mqtt_discovery
